All the shortcuts:
  1. Ctrl + Space (Show suggestions)
  2. Ctrl + L (Select the whole line)
  3. Ctrl + P (Shows the search bar)
  4. Ctrl + B (Toggles the sidebar)
  5. Ctrl + / (Toggles line comment)
  6. Ctrl + O (Opens the current file in the file explorer)
  7. Ctrl + X (Deletes the line)
  8. Ctrl + D (One by one selection)
